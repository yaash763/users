This code is developed using Spring boot and Java 8

and To test the API's used Swagger API

steps to run the application
1. build the application using below command
 mvn clean install

 2. run as java application
   or run Application.java 
   
   default it runs on 8080 port
   
 3. use the below url to access swagger UI to test 
 
  http://localhost:8080/swagger-ui.html 
 
   OR
   
   run the below API from browers to get allusers with project data
    http://localhost:8080/api/allusers
   
   
 4. Run UserControllerTest.java as Junit to execute test cases.
